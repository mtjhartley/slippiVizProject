Welcome to StatsGG, a data visualization project created by Michael Hartley using the smash.gg API. This data was taken from Smash Summit Spring 2017, a tournament hosted by Beyond the Summit. Analysis was done on the data to find values that are (hopefully) interesting. The tables are interactive; clicking a row send that video to the punish! Enjoy!
